# 🌱 Ecosia Search Engine Redesign (HCI Project)

## Project Overview

This project focuses on redesigning the Ecosia mobile application to improve usability, user experience, and functionality while supporting Ecosia's environmental sustainability mission.

---

## Problem Statement

Several usability issues were identified in the existing Ecosia application:

- Limited search result diversity
- Text-heavy user interface
- Lack of integrated map functionality
- Limited image search capabilities
- Inefficient search history management

---

## Target Users

### 👨‍💼 Online Researcher
- Needs accurate location search and navigation
- Requires real-time traffic information
- Needs route optimization features

### 👨‍🎓 Student
- Frequently searches for images for assignments
- Requires high-quality image results
- Needs a simple and user-friendly interface

### 🌳 Environmental Enthusiast
- Frequently revisits previously viewed content
- Requires easy access to search history
- Interested in environmental-related information

---

## Main Features Developed

### 📍 Search Location (EcoMap)

- Integrated map and navigation system
- Multiple map views:
  - Standard
  - Satellite
  - Terrain
- Transport mode selection
- Route guidance and directions

### 📸 Image Searching

- Search using uploaded images
- Camera capture support
- Gallery image upload
- High-definition (HD) image option
- Related image recommendations

### 📚 Search History

- View previous searches
- Search history filtering
- Calendar-based history search
- Delete selected history entries
- Categorized browsing history

---

## Design Process

The project followed a User-Centered Design (UCD) approach:

1. User Analysis
2. Persona Development
3. Task Analysis
4. Storyboard Design
5. Alternative Sketch Design
6. Wireframe Development
7. Interactive Prototype Design
8. Usability Testing
9. Improvement and Refinement

---

## HCI Principles Applied

### Shneiderman's Golden Rules

- Strive for Consistency
- Offer Informative Feedback
- Enable Frequent Users to Use Shortcuts
- Support Internal Locus of Control
- Reduce Short-Term Memory Load
- Design Dialogs to Yield Closure

### Gestalt Principles

- Similarity
- Proximity
- Enclosure
- Figure-Ground

## Usability Goals

### Effectiveness
- Users can complete tasks accurately.
- Search results match user expectations.

### Efficiency
- Tasks can be completed with minimal effort.
- Quick access to frequently used functions.

### Memorability
- Easy to remember interface layout.
- Minimal learning curve.

### Satisfaction
- User-friendly design.
- Visually appealing interface.

## Usability Testing

Three key tasks were evaluated:

### Task 1: Search Location
✅ Users successfully searched locations and obtained directions.

### Task 2: Image Searching
✅ Users successfully searched images using uploaded photos.

### Task 3: Search History
✅ Users successfully accessed and managed browsing history.
