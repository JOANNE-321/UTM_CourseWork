# 🎯 Summary of What I Have Gained from This Course

The Programming Technique II course provided valuable understanding and practical experience in Object-Oriented Programming (OOP) using the C++ programming language. Throughout the semester, I learned how to design software using objects and classes while applying important OOP principles such as encapsulation, inheritance, polymorphism, and associations.

The course emphasized both theoretical concepts and practical implementation through programming exercises and project-based learning activities. These experiences strengthened my programming skills and improved my ability to develop structured and reusable software solutions.

## 📚 Library Management System Project

The major project for this course involved developing a **Library Management System** using C++. The project required the application of object-oriented design principles to create a functional system for managing library-related operations.

### Key Learning Activities

- Object-oriented system design
- Class and object implementation
- UML class diagram interpretation
- Modular programming
- Software testing and debugging
- Documentation and report preparation

### Skills Developed

- Object-Oriented Programming (OOP)
- C++ programming
- Problem-solving and logical thinking
- Software design and development
- Debugging and testing
- Teamwork and collaboration
- Technical documentation

### Key Takeaway

The project demonstrated how object-oriented concepts can be applied to develop structured, maintainable, and scalable software systems. It also strengthened my understanding of how software design influences system quality and maintainability.
