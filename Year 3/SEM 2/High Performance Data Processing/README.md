# 🎯 A. Summary of What I Have Gained from This Course

## 📄 Assignment

### Assignment 1: Academic Writing – Databricks Lakehouse

Through this assignment, I gained a deeper understanding of the Lakehouse architecture and how it combines the strengths of both data lakes and data warehouses. I also improved my academic writing, literature review, critical analysis, and technical reporting skills.

### Assignment 2: Mastering Big Data Handling

This assignment provided hands-on experience in handling large datasets using different big data processing frameworks. I learned techniques for processing, cleaning, and optimizing datasets containing millions of records.

**Key Learning Outcomes**

- Understanding big data processing concepts
- Data cleaning and transformation
- Performance optimization techniques
- Comparing different big data frameworks
- Efficient handling of large-scale datasets

---

## 🌐 Project Phase 1: Optimizing High-Performance Data Processing for Large-Scale Web Crawlers

This project involved developing a high-performance web crawler capable of collecting more than 100,000 records from a Malaysian website. My team applied optimization techniques such as multithreading, multiprocessing, and distributed processing to improve crawling performance and efficiency.

**Key Learning Outcomes**

- Web crawling and large-scale data collection
- Data cleaning and transformation
- High-performance computing concepts
- Performance optimization techniques
- Performance evaluation and comparison
- Team collaboration and project management

**Key Takeaway**

This project helped me understand how high-performance computing techniques can significantly improve the efficiency and scalability of data collection systems.

---

## ⚡ Project Phase 2: Real-Time Sentiment Analysis Using Apache Spark and Kafka

This project focused on building a real-time sentiment analysis pipeline using Apache Kafka and Apache Spark. The system collected text data, processed it in real time, classified sentiment, and visualized the results through dashboards.

**Key Learning Outcomes**

- Apache Kafka for real-time data streaming
- Apache Spark for large-scale data processing
- Natural Language Processing (NLP)
- Sentiment analysis model development
- Real-time analytics pipeline design
- Data visualization and reporting

**Key Takeaway**

This project provided practical experience in developing modern real-time data processing systems that are widely used in industry.
