# 🎯 A. Summary of What I Have Gained from This Course

## Overview

The Database course provided fundamental knowledge and practical experience in database design, database development, and database management. Throughout the semester, I learned how to design relational databases, create database schemas, write SQL queries, and implement database solutions to support information systems.

---


## 📚 Assignments

The assignment provided additional practice in database concepts and SQL implementation.

### Key Learning Outcomes

- SQL query writing
- Data retrieval and manipulation
- Database problem-solving
- Query optimization concepts

---

## 🧪 Laboratory Activities

### Lab 1: Create Database

This lab introduced the process of creating databases, tables, relationships, and constraints using SQL.

### Lab 2: SQL Query Exercises for Online Bookstore Database

This lab focused on writing SQL queries to retrieve, filter, sort, and analyze data from a database system.

### Skills Developed

- Database creation
- Table design
- SQL commands
- Data retrieval
- Query development

### Key Takeaway

The laboratory sessions provided hands-on experience with SQL and helped strengthen my understanding of database operations.

---

## 💻 Project: KADA System Database Design

The major project for this course involved designing the database for the KADA System. This project required the application of database design principles to develop a structured and efficient database that supports system requirements.

### Key Learning Outcomes

- Database requirement analysis
- Entity Relationship Diagram (ERD) design
- Relational database modelling
- Database normalization
- Primary and foreign key implementation
- Database documentation

### Skills Developed

- Database design
- Logical data modelling
- Problem-solving
- Team collaboration
- Technical documentation

### Key Takeaway

The KADA System project provided valuable experience in designing a real-world database solution and demonstrated the importance of proper database design in supporting system functionality and data integrity.

# 💡 B. Suggested Improvements

## More Real-World Database Case Studies

Additional industry-based database projects would help students better understand how database systems are implemented and managed in real organizations.

## More Advanced SQL Exercises

Including more complex SQL scenarios involving stored procedures, triggers, and optimization techniques would further enhance practical database skills.

# 🌟 C. Personal Reflection

This course provided a strong foundation in database concepts and database development. Through the laboratory activities, assignments, assessments, and the KADA System project, I gained practical experience in designing and implementing relational databases.

The most valuable experience was working on the KADA System database design project because it allowed me to apply theoretical concepts to a realistic system development scenario. I learned how business requirements are translated into database structures and how proper database design contributes to system efficiency and data integrity.

In addition, the course strengthened my SQL skills, analytical thinking, and problem-solving abilities. The hands-on activities helped me become more confident in working with databases and managing structured data.

Overall, this course enhanced both my technical knowledge and practical experience in database design and development, which are essential skills for future software engineering and data engineering projects.

## 🚀 Conclusion

Overall, the Database course was a valuable learning experience that provided a strong understanding of database concepts, SQL programming, and database design principles. The knowledge and skills gained from this course serve as an important foundation for future studies and projects involving data management and information systems.
