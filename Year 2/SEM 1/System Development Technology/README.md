# 🎯 A. Summary of What I Have Gained from This Course

## Overview

The System Development Technology course provided practical exposure to web application development using modern web technologies and development frameworks. Throughout the course, I learned how to design, develop, and implement web-based systems using HTML, PHP, Bootstrap, and the Model-View-Controller (MVC) architecture. The course emphasized both frontend and backend development while introducing industry practices in system development.

# 🌟 C. Personal Reflection

This course significantly improved my understanding of web application development and system implementation. Before taking this course, I had limited experience in developing complete web-based systems. Through the assignments, laboratories, and KADA Project, I learned how frontend and backend technologies work together to create functional applications.

The laboratory activities provided practical experience in HTML, PHP, and Bootstrap, while the MVC notes introduced me to software architecture concepts that are widely used in modern system development. These activities strengthened my technical skills and improved my ability to develop structured and maintainable applications.

The KADA Project was the most valuable part of the course because it allowed me to apply theoretical concepts to a realistic development scenario. Through this project, I improved my teamwork, communication, problem-solving, and project management skills while gaining experience in the complete software development lifecycle.

Overall, this course enhanced my technical competency in web development and system design while providing valuable practical experience that will support future software engineering and application development projects.

## 🚀 Conclusion

Overall, System Development Technology was a highly practical course that provided valuable experience in web development, system architecture, and application implementation. The knowledge and skills gained from this course established a strong foundation for future software engineering and system development projects.
