# 🎯 A. Summary of What I Have Gained from This Course

## Overview

The Object Oriented Programming (OOP) course provided a strong foundation in object-oriented software development using Java. Throughout the semester, I learned how to design and implement software systems using core OOP concepts such as classes, objects, encapsulation, inheritance, polymorphism, abstraction, constructors, method overloading, and access modifiers.

The course emphasized both theoretical understanding and practical implementation through laboratory exercises and programming activities.

---

## 🧪 Laboratory Exercises

### Lab 1: Object-Oriented Programming Fundamentals

This lab introduced the core concepts of object-oriented programming using Java.

#### Key Learning Outcomes

* Class and object creation
* Instance variables and methods
* Constructors
* Getters and setters
* Encapsulation
* Access modifiers
* Static and non-static members
* Method overloading

---

### Lab 2: Java Arrays and Problem Solving

This lab focused on array manipulation, debugging, and algorithm implementation using Java.

#### Key Learning Outcomes

* Array declaration and initialization
* Array traversal techniques
* Error identification and correction
* Helper method development
* Logical problem solving

---

### Lab 3: Share Store System Development

This lab required the implementation of a complete object-oriented system consisting of Store, SpaceLot, Member, and Item classes.

#### Key Learning Outcomes

* Object-oriented system design
* Association and aggregation relationships
* Object referencing
* List management
* Class interaction and collaboration

#### Key Takeaway

This activity demonstrated how multiple classes can work together to form a complete information system using object-oriented principles.

---

### Lab 4: School Management System

This lab focused on modelling class relationships within a School Management System.

#### OOP Concepts Applied

* Association
* Aggregation
* Composition
* Inheritance
* Polymorphism

#### Key Learning Outcomes

* Class relationship modelling
* Method overriding
* Inheritance hierarchy design
* Polymorphic behavior
* Software architecture concepts

#### Key Takeaway

This exercise strengthened my understanding of how real-world systems are represented through object-oriented models.

---

### Lab 5: Error Detection and Code Completion

This lab involved identifying programming errors, completing missing statements, and debugging Java programs.

#### Key Learning Outcomes

* Debugging techniques
* Error analysis
* Program testing
* Code correction
* Software maintenance

#### Key Takeaway

The activity improved my ability to identify issues, troubleshoot programs, and produce more reliable software solutions.

Additional case studies and larger projects would provide students with greater exposure to practical software development scenarios.

Introducing concepts such as graphical user interfaces (GUI), database connectivity, and design patterns would further enhance students' programming skills.

# 🌟 C. Personal Reflection

This course significantly improved my programming and problem-solving abilities. Before taking this course, I mainly focused on procedural programming concepts. Through the laboratory exercises and system development activities, I learned how object-oriented programming helps organize software into reusable, maintainable, and scalable components.

The Share Store System and School Management System exercises were particularly valuable because they allowed me to apply theoretical concepts such as inheritance, polymorphism, association, aggregation, and composition in practical scenarios. These activities helped me better understand how real-world software systems are designed and developed.

In addition, the course strengthened my logical thinking, debugging skills, and ability to analyze software requirements before implementation. I also became more confident in designing classes and developing structured Java applications.

Overall, this course enhanced both my theoretical understanding and practical competency in object-oriented programming and software development.

## 🚀 Conclusion

Overall, Object Oriented Programming was a highly valuable course that provided a strong foundation in Java programming, object-oriented design, and software development principles. The knowledge and skills gained from this course will be beneficial for future software engineering, system development, and application development projects.
